# avatar-picker

### 简介

`avatar-picker`是一个头像和昵称选择/填写组件。<u>可以方便的实现用户的头像选择和昵称选择与填写</u>。此外还有配套的java后端代码实现头像的存储，昵称的存储。

### 适用场景

- 微信小程序

> 其它端的我暂时没试过，后续也会持续更新

### 使用步骤
[Dcloud插件市场](https://ext.dcloud.net.cn/plugin?id=11935)
插件市场导入，示例工程中有详细操作
